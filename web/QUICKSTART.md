# 🚀 Быстрый запуск AmneziaWG Web Interface

## Что это?

Современный веб-интерфейс для управления AmneziaWG VPN сервером с красивым UI и real-time мониторингом.

**Стек технологий:**
- **Frontend**: React 18 + TypeScript + Chakra UI v3 + Charts
- **Backend**: Go + Gin + WebSocket для real-time обновлений
- **Infrastructure**: Docker + Nginx + автоматические деплойменты

## 📋 Возможности

✅ **Dashboard** - красивая панель с графиками подключений и трафика  
✅ **Client Management** - добавление/удаление клиентов, QR коды, экспорт конфигов  
✅ **Server Settings** - настройка обфускации, сетевых параметров  
✅ **Real-time Logs** - потоковые логи через WebSocket  
✅ **Responsive Design** - адаптивный дизайн для мобильных устройств  
✅ **Dark/Light Theme** - переключение темы  

## ⚡ Быстрый запуск (1 команда)

```bash
# Переходим в директорию веб-интерфейса
cd web

# Запускаем весь стек одной командой
make up
```

**Готово!** 🎉

- 🌐 **Веб-интерфейс**: http://localhost
- 📡 **API**: http://localhost:8080

## 🛠️ Разработка

### Установка зависимостей
```bash
make install
```

### Режим разработки
```bash
# Терминал 1 - Backend
make dev-backend

# Терминал 2 - Frontend  
make dev-frontend
```

Фронтенд будет доступен на http://localhost:3000 с hot reload.

## 📊 Управление

### Основные команды
```bash
make up           # Запустить весь стек
make down         # Остановить все
make restart      # Перезапустить
make status       # Показать статус
make logs         # Показать логи
```

### Веб-интерфейс отдельно
```bash
make web-up       # Только веб-интерфейс
make web-down     # Остановить веб-интерфейс
make web-logs     # Логи веб-интерфейса
```

### Разработка
```bash
make build        # Собрать проект
make test         # Запустить тесты
make clean        # Полная очистка
```

## 🐳 Docker Services

Запускаются следующие контейнеры:

| Сервис | Порт | Описание |
|--------|------|----------|
| `amneziawg-web-nginx` | 80 | Nginx reverse proxy |
| `amneziawg-web-frontend` | 3000 | React приложение |
| `amneziawg-web-api` | 8080 | Go API сервер |
| `amneziawg-server` | 51820 | AmneziaWG VPN сервер |

## 🎯 Что умеет веб-интерфейс?

### 📈 Dashboard
- Статус сервера в реальном времени
- Количество подключенных клиентов
- Графики трафика и подключений
- Кнопки управления сервером (старт/стоп/рестарт)

### 👥 Управление клиентами
- Список всех клиентов с статусом подключения
- Добавление новых клиентов (с автоматическим или ручным IP)
- Генерация QR кодов для быстрого подключения
- Экспорт конфигураций клиентов
- Удаление клиентов

### ⚙️ Настройки сервера
- Базовая конфигурация (порт, сеть, DNS)
- Настройка параметров обфускации
- Готовые профили обфускации (Standard, Enhanced, DNS Simulation)
- Сохранение изменений в реальном времени

### 📝 Логи
- Просмотр логов сервера
- Real-time обновления через WebSocket
- Фильтрация по уровням (error, warn, info, debug)
- Поиск по тексту логов
- Экспорт логов в файл

## 🔧 Интеграция с существующим проектом

Веб-интерфейс полностью интегрирован с существующим AmneziaWG проектом:

- Использует те же **make команды** для управления сервером
- Читает и модифицирует тот же **.env файл**
- Работает с теми же **конфигурационными файлами**
- Показывает **реальные данные** из контейнера AmneziaWG

## 🚨 Решение проблем

### Веб-интерфейс не открывается
```bash
make status       # Проверить статус всех сервисов
make web-logs     # Посмотреть логи
```

### API не отвечает
```bash
make web-logs-api     # Логи API сервера
curl http://localhost:8080/health  # Проверить health check
```

### Проблемы с AmneziaWG сервером
```bash
cd .. && make status  # Статус основного сервера
cd .. && make logs    # Логи AmneziaWG
```

### Полная переустановка
```bash
make clean        # Удалить все
make up           # Заново собрать и запустить
```
